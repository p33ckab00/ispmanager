Phase 1 Rules

  • One subscriber can have only one active Premium NMS mapping at a time.
  • The subscriber module stays the billing and account source of truth.
  • Saving in Premium NMS mirrors the node summary back to the subscriber page.
  • Basic node reassignment is locked once a Premium NMS mapping exists.
  • Only eligible endpoints are shown per selected node, and occupied endpoints are excluded from reassignment lists.
  • Structured cable core assignments reserve or use one core at a time and sync back to cable inventory.
